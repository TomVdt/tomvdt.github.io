# Français
activité à faire pendant l'heure de cours aujourd'hui : préparation de cet oral.
Vous avez tous reçu en travail à faire pour lundi 23 mars le sujet suivant :
Oral : Quel effet la lecture de la fin de L’usage du monde produit-elle sur vous ? Justifiez en développant trois arguments bien construits. Vous formulerez votre thèse qui explicitera l'effet produit sur vous et développerez trois arguments selon le schéma vu en classe (en vous appuyant sur le texte).
Je vous rappelle que le schéma est :
1- affirmation,
2- justification ( avec une proposition causale )
3 - un exemple développé qui s'appuie sur un passage du texte commenté.
Durant cette séance, vous allez formuler votre thèse et rechercher vos arguments illustrés d'un exemple par écrit. Puis vous les reprendrez pour enregistrer oralement votre travail sur le fichier qui suit le document envoyé en travail à faire. Bon courage !
